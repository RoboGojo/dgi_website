# Delta Gamma Iota Website

A Flutter-based website for the Delta Gamma Iota fraternity.

## Features

- **Home page**
- **Cardinal Beliefs**
- **Banner & Navigation Bar**
- **Footer**

## Prerequisites
- [Visual Studio Code](https://code.visualstudio.com/download)
- [Flutter SDK](https://flutter.dev/docs/get-started/install) (3.10.0 or higher)
- [Dart SDK](https://dart.dev/get-dart) (comes with Flutter)
- A web browser (Edge or Chrome recommended for development)

## Getting Started

### 1. Clone the Repository
In VS Code, press <kbd>Ctrl</kbd> + <kbd>J</kbd> to open the terminal.

In the terminal, run the following:
```bash
git clone <repository-url>
cd dgi_website
```

### 2. Install Dependencies
```bash
flutter pub get
```

### 3. Verify Flutter Installation
```bash
flutter doctor
```
Make sure all required dependencies are installed and configured correctly.

### 4. Run the Application
To run the website in development mode:
```bash
flutter run --target lib/main_development.dart
```

To run the website in production mode:
```bash
flutter run --target lib/main_production.dart
```
